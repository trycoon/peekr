Peekr

Bildgissningsspel med 80-tals touch.

Download .zip Download .tar.gz View on GitHub

Instruktioner

Gissa vem som döljer sig bakom kikhålet, flytta runt kikhålet med hjälp av muspekaren eller fingret. Skriv din gissning i inmatningsfältet under bilden och tryck på knappen för att lämna ditt svar. Om du gissade fel så får du flera försök, fast ett rätt svar ger dig lägre antal poäng och ett större kikhål som visar större del av bilden. Spelet är över om du inte har gissat rätt på 1 poängsnivån. Om du svarat rätt så kan du gå vidare och gissa på en ny bild.

Installation

git clone https://github.com/trycoon/peekr.git
cd peekr
npm update
npm install -g yo
npm install -g generator-backbone
npm install
bower install

Starta applikation in utvecklingsläge

grunt serve
